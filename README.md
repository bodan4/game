# Space Shooter Game

A simple browser-based 2D space shooter built with vanilla JavaScript, HTML, and CSS.

This is a small interactive game where the player controls a ship to avoid and shoot asteroids in space.

## Demo

You can run the game locally by opening `index.html` in your browser.

## Features

- Player spaceship with movement controls
- Randomly spawning asteroids
- Collision detection
- Scoring system
- Simple gameplay loop

## Tech Stack

- JavaScript — game logic and mechanics
- HTML — game structure
- CSS — styling
- Images and assets in repository

## How to Play

1. Open `index.html` in your web browser
2. Control the ship using arrow keys or WASD and SPACE for shooting asteroids
3. Avoid or shoot asteroids
4. Survive as long as possible to increase score
